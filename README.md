# 🧸 Online Teddy Bear Store (Team Project)

> **Score: 8.0/10** - Website bán gấu bông trực tuyến xây dựng trên nền tảng PHP thuần và kiến trúc MVC.

## 🌟 Giới thiệu
Dự án tập trung vào việc xử lý luồng mua hàng cơ bản (E-commerce Flow) và quản lý dữ liệu bằng MySQL. Đây là dự án giúp mình nắm vững nền tảng về lập trình web và tư duy hệ thống.

## 💻 Công nghệ sử dụng
* **Language:** PHP.
* **Database:** MySQL (phpMyAdmin).
* **Frontend:** HTML5, CSS3, JavaScript.
* **Architecture:** MVC (Model-View-Controller) cơ bản.

## ✨ Chức năng chính
* **Sản phẩm:** Hiển thị danh sách sản phẩm theo danh mục, chức năng tìm kiếm sản phẩm.
* **Giỏ hàng:** Thêm/sửa/xóa sản phẩm trong giỏ hàng, tính tổng tiền.
* **Người dùng:** Đăng ký, đăng nhập hệ thống và quản lý lịch sử đơn hàng.
* **Admin:** Giao diện quản lý sản phẩm (CRUD) và xem danh sách đơn hàng khách đã đặt.

## 🛠️ Hướng dẫn cài đặt
1. **Database:** Export file `.sql` kèm theo vào phpMyAdmin.
2. **Cấu hình:** Chỉnh sửa thông tin kết nối Database trong file `config.php` (hoặc file kết nối tương ứng của bạn).
3. **Run:** Sử dụng XAMPP/Laragon để chạy dự án trên Localhost.
